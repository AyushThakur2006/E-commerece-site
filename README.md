# Day-1

Installation for the backend 
- list of installations :-
1. express - web framwork used to build API & handling requests,responses and routes
2. mongoose - Helps to connect node.js with mongodb database
3. bcryptjs - Hashes passwords and important info before sending to the database
4. jsonwebtoken - Used to remember that the user is logged in 
5. cookie-parser - Reads saved login data from the browser
6. cors - Allows frontend (React) to access backend (Node)
7. dotenv - Keeps passwords and secret keys hidden (act as a Secret Locker)
